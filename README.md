# CoffeeURL
A URL shortener written in Python 3.

## Requirements
This project requires the following libraries:
 - sqlite3
 - flask
 - gevent
 
## Usage
Simply run:
```bash
$ python3 main.py
```
/!\ The default port is 8080.
